\begin{DoxyItemize}
\item COO\+: coordinates uncompressed, mapping indices to values
\item CSR\+: compressed sparse row
\item CSC\+: compressed sparse column
\end{DoxyItemize}

\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Additional instructions}{Additional instructions}}\label{index_autotoc_md3}
If you want to read a full matrix you can set a threshold for considering a number as zero, thus not adding it as an element of the matrix.

To do so you can edit the {\ttfamily Makefile} adding a new flag to the compiler\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{CPPFLAGS\ +=\ -\/D\ ZERO\_TOL=1e-\/8}

\end{DoxyCode}


If the variable {\ttfamily ZERO\+\_\+\+TOL} is not redefined, the default value is {\ttfamily 1e-\/06}.\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Testing}{Testing}}\label{index_autotoc_md4}
