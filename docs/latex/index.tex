\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Additional instructions}{Additional instructions}}\label{index_autotoc_md4}
If you want to read a full matrix you can set a threshold for considering a number as zero, thus not adding it as an element of the matrix.

To do so you can edit the {\ttfamily Makefile} adding a new flag to the compiler\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{CPPFLAGS\ +=\ -\/D\ ZERO\_TOL=1e-\/8}

\end{DoxyCode}


If the variable {\ttfamily ZERO\+\_\+\+TOL} is not redefined, the default value in the header source code is {\ttfamily 1e-\/08}.\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Testing}{Testing}}\label{index_autotoc_md5}
In the {\ttfamily main.\+cpp} file is possible to test different sections of the cody by changing the values from {\ttfamily false} to {\ttfamily true} in the {\ttfamily if} statements.
